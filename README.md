scanraspberrypi
===============

在自己的机器上运行，就可以知道同网段中树莓派的ip地址。

use:
====

```
npm install scanraspberrypi
```

example:
--------

```
var scanraspberrypi=require('scanraspberrypi');
scanraspberrypi();
```
